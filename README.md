# Donald

Donald is a personal, always-on AI operator — a chief-of-staff + analyst + execution desk
for three things:

1. **Research** — deep, multi-source, fact-checked analysis (far beyond a single chat answer).
2. **Trading** — supervised and (eventually) autonomous trading off your signals, with hard safety rails.
3. **Business & Personal OS** — email, outreach, content, docs, and comms — the base layer everything routes through.
